# crypto-news-parser
Парсер парсит 2 сайта:

https://forklog.com/

https://cryptonews.net/ru/


# что делает
Собирает ссылки на статьи.

Заголовки статей.

Получает ссылку на изображение-превью статьи.

Сохраняет данные в JSON в следующем формате:
```json
[
    {
        "url": "ссылка на статью",
        "text": "заголовок статьи",
        "img_url": "ссылка на превью статьи"
    }
]
```
# Важно

Структура сайтов может изменяться, поэтому парсер может перестать работать.

Все данные сохраняются локально в JSON.

# Как использовать

Склонировать репозиторий:

```bash
git clone https://github.com/BlackRaisons/crypto-news-parser.git
```
перейти в папку проекта:

```bash
cd ~/ваш/путь/до/папки/crypto-news-parser
```
Установить зависимости:

```bash
pip install -r requirements.txt
```
Запустить скрипт:

```bash
python main.py
```
